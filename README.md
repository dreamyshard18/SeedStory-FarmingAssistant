## рҹҢҝ About the Project

There's something incredibly rewarding about growing your own little patch of green.

Whether it's vegetables on a balcony, herbs in the kitchen, or a thriving backyard garden, every plant deserves the right care. SeedStory was created to make that journey easier by providing reliable, AI-powered farming guidance tailored to Kerala's unique environmental conditions.

From heavy monsoons to humid summers, the assistant understands regional challenges and offers practical recommendations for plant care, cultivation, soil health, seasonal planning, pest management, and more.

Unlike general-purpose chatbots, **SeedStory is intentionally focused only on farming and gardening**, ensuring responses stay relevant, accurate, and useful.

Think of it as your digital farming companion. рҹҢұ

## вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/dreamyshard18/SeedStory-FarmingAssistant.git
```

Move into the project

```bash
cd SeedStory-FarmingAssistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=your_api_key_here
```

Run the application

```bash
python app.py
```
